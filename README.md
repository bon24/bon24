## Projects

### 서울형 키즈카페 입지분석 프로젝트

공공데이터와 입지 최적화 모델을 활용하여 광진구 내 서울형 키즈카페 신규 입지 후보지를 도출한 프로젝트입니다.

- 20개 이상의 공공데이터 수집 및 전처리
- 아동 수요를 반영한 가중치 K-means 적용
- 수요지, 후보지, 기존시설을 분리한 LSCP 입지 최적화
- 신규 입지 후보 6곳 제안
- 아동복지 사각지대 비율 8.5% → 1.7% 감소 확인
